01 — Capabilities

Six Process. One closed loop.

Nothing escapes - every action and every decision, fully visible from intent to outcome.

Action01 / 06
Ingest evidence
Push CDR batches, billing extracts, signaling logs or IoT charge records into the encrypted GCS bucket. Signed uploads, SHA-256 verified on arrival.
Caller: operator · portal
Action02 / 06
Dispatch agents
Wakes Revenue Hawk and Fraud Interceptor against the newest partition. Triggers automatically on bucket write — or manually on demand.
Caller: cloud function · auto
View03 / 06
Query findings
Filter by severity, category, country, tenant, or window. Cursor-paginated. Every row carries a rule hash and the evidence pointer.
Caller: operator · console
View04 / 06
Agent history
Complete, append-only record of every agent action: which rule fired, what data it read, what it proposed, and who approved. Built for regulators.
Caller: compliance · export
View05 / 06
Health summary
Live counters for the operations centre: open anomalies by severity, last scan timestamps per agent, queue depth and model version pinned in production.
Caller: dashboard · 30s poll
Action06 / 06
Ask the agent
Free-text investigation: "show SIM swaps in Iberia last week." The agent decomposes the query, runs tool calls, and returns an annotated answer with citations.
Caller: analyst · chat
